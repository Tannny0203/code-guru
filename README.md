# CodeMentor AI

## 🚀 Quick Start — 3 steps

### Step 1 — Open `.env` and paste your 3 values

```env
GEMINI_API_KEY=your_gemini_key_here
HINDSIGHT_API_KEY=your_hindsight_key_here
HINDSIGHT_BASE_URL=https://your-instance.hindsight.vectorize.io
```

| Key | Where to get it |
|-----|----------------|
| `GEMINI_API_KEY` | https://aistudio.google.com/app/apikey |
| `HINDSIGHT_API_KEY` | https://app.hindsight.vectorize.io → Settings → API Keys |
| `HINDSIGHT_BASE_URL` | https://app.hindsight.vectorize.io → your instance URL |

### Step 2 — Install dependencies
```bash
npm install
```

### Step 3 — Run
```bash
npm run dev
```
Open **http://localhost:3000**

> If any key is missing the server will tell you exactly which one to fix before starting.
